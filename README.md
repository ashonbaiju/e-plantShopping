# Paradise Nursery Shopping Application

Paradise Nursery is an online plant shop web application built using React and Redux.
Users can browse different categories of houseplants, view plant details, add plants to a shopping cart,
update quantities, remove items, and view the total cost dynamically.

## Features
- Browse plants by category
- Add and remove plants from cart
- Increase and decrease quantity
- Dynamic cart total
- Responsive UI

## Technologies Used
- React
- Redux Toolkit
- Vite
- CSS
