# e-plantShopping

## Paradise Nursery Shopping Application

e-plantShopping is a React-based shopping cart web application for an online plant shop called Paradise Nursery.
The application allows users to browse different categories of houseplants, view plant details, add plants to the shopping cart,
update quantities, remove items, and see the total cost dynamically.

## Features
- Browse plants by category
- Add plants to cart
- Increase or decrease item quantity
- Remove items from the cart
- Dynamic cart total calculation
- User-friendly navigation

## Technologies Used
- React
- Redux Toolkit
- Vite
- JavaScript
- CSS
